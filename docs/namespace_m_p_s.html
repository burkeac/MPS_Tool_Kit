Input code values should be linear Rec2020 values ie. PQ_RGB = 0-10,000
[In] Rec2020 R CV
[In] Rec2020 G CV
[In] Rec2020 B CV
[In] PQ = true, HLG = false
[In] Scale the values to JND
[Return] std::vector<float> in I, Ct, Cp format
Implementation based on Dolby White Paper "Perceptual Color Volume" and WHAT IS ICTCP? https://www.dolby.com/us/en/technologies/dolby-vision/measuring-perceptual-color-volume-v7.1.pdf https://www.dolby.com/us/en/technologies/dolby-vision/ictcp-white-paper.pdf
