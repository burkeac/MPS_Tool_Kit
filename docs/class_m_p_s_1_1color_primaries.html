Detailed Description

Defines the primaries of a pre-defined or custom colorspace i.e. Rec709 or Rec2020.

Provides easy access and storage to predefined and custom primary sets Methods and members allow for easy modification of selected primarie sets. Data stored in this class should be in CIE xy as outlined in the CIE 1931 Color Space
