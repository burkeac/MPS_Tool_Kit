Generates the PM and Inverse PM from the set of primaries.

Calculates the forward and inverse PMs of the given primary sets
[In] colorPrimaries the primaries the phosphore matrix should be calcualted for. Passed by reference
[In] The acutal luminance the PM should be calculated for. Default = 100 nits
[In] The aim luminance the PM should be calculated for. Default = 100 nits
[In] Set the private PM members of the object. Default = True.
